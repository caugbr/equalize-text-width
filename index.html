equalizeTextWidth()

iguala a largura do texto mudando o tamanho da fonte

Este é um plugin jQuery direcionado basicamente a designers e sua idéia é muito simples: igualar a largura do texto em um elemento mudando o tamanho da fonte linha a linha, até que todas as linhas de texto tenham o tamanho mais próximo possível da largura do elemento que contém o texto.

Se enviarmos o parâmetro 'text', caso haja uma palavra que sozinha seja maior que a linha, a largura levada em consideração será a da palavra, caso contrário ela será reduzida para caber no elemento. Há um segundo parâmetro boleano que diz se, caso o primeiro parâmetro seja 'text', se o elemento deve crescer para acompanhar o tamanho do texto (apenas o primeiro exemplo pode mostrar isso tudo). Usando esse parâmetro, as quebras de linha originais podem não ser seguidas, pois o elemento será renderizado numa largura diferente.

Usada com o parâmetro 'undo', o sistema desfaz o que foi feito.
